## 🚀 核心项目

### 🐾 毛孩疫苗本 — AI宠物健康管理工具
**从痛点洞察 → A/B实验验证 → 全量发布，展现完整产品闭环**

- **用户洞察**：访谈 12 位宠物主，发现“提醒不灵活、导出缺失”为高频痛点
- **核心方案**：离线 OCR 疫苗识别（准确率 40%→80%+）、按宠物独立提醒、稍后/忽略策略、品牌/批号追溯
- **A/B 测试**：实验组（灵活提醒）vs 对照组，提醒设置转化率 26%→42%（+61.5%），7日留存 31%→44%（+41.9%），统计显著（p<0.05）
- **迭代亮点（v1.8）**：
  - 数据看板与年度报告（花费趋势/疫苗覆盖率/体重变化）
  - 体重独立管理 + 月历视图（可点击某天弹出当日到期清单）
  - 日历清单内支持“已预约/取消预约”双态切换，直接同步提醒转化统计
  - 多宠对比表新增“本月花费/超期项”，健康评分用于风险可视化
  - PWA 能力完善（manifest + service worker + 系统通知）
  - UI 全面响应式重构（手机/平板/PC）与多主题风格
- **商业收益**：开发成本 3 人天，月增量收益 4500 元，6 个月累计 ROI >300%
- **用户原声**：“家里两只狗终于能分开提醒了，不会再把A狗的针记到B狗头上，这个真的救命。” —— 双狗家庭用户·Jason

### 🎙️ 会议·反刍舱 — AI 会议复盘与待办管理
**打造“录音→转写→待办→反刍”闭环，解决会议内容流失**

- **产品逻辑**：浏览器端运行 Whisper 转文字，DeepSeek/本地模型提取待办，零服务端成本
- **数据验证**：上线反刍提醒后，7日待办完成率从 52% 提升至 71%（+19pp），平均完成周期缩短 1.8 天
- **迭代亮点（v1.4）**：
  - 待办支持状态流转（待办/进行中/阻塞/完成）与历史回写
  - 历史会议支持关键词+任务质量筛选（含超期识别）
  - 新增会议健康评分（A/B/C）与近7天KPI卡片
  - 周报/月报支持 TOP 关键词，汇报可直接复制输出
  - UI 完成手机/平板/PC 响应式优化
- **用户原声**：“以前会后待办都散在聊天记录里，现在我每天只看这一个页面就知道谁卡住了。” —— 项目经理·杨

### ✍️ 星途·创作舱 — 网文创作者效率工具
**解决“大纲、角色、灵感四处分散”的创作管理痛点**

- **功能整合**：Markdown 大纲编辑器 + 角色/世界观面板 + 800+ 标题灵感库 + 每日写作目标
- **迭代亮点（v1.7）**：

  - 全文搜索（章节名/角色/标签/参考书目）+ 时间线视图
  - 创作健康评分（含等级）+ 多项目对比看板
  - 周报/月报导出（含 TOP 关键词）
  - 版本快照与回滚 + 本地点评锚点 + PWA 离线缓存
- **体验细节**：多主题切换（含墨水屏模式），UI 与交互细节统一重构
- **用户原声**：“我现在真的是先开这个再写文，大纲、人物、标题都在一起，进入状态快很多。” —— 签约作者·miki
