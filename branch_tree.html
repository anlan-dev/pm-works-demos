《雾灯残页》游戏结构示意图

Loop Flow · 树状分支总览 · Philosophical Bias Axes · Ending Map

命运轴：接受既定／改写条文／或被反噬缠着——都有价码，没有白捡的顺境。 个体轴：保全自保、或以磨损换照应、一路溶解到记不清边界，三条里选一条就得认一条疼。 未知轴：拥抱不可测、借势利用未知、抑或拒绝承认——都不是道德判决，只是立场。 无绝对好坏，只有偏向与代价 每个结局都必须给收益 + 延迟后果

《story_text》三律 · 叙事方法论备忘（与 game/script.rpy / Ren’Py 开场一致）

  • 信息的真与假：告示口传与经卷常掐架；与其说哪份是「铁板钉钉」，不如说玩家在哪种疼里肯采信哪一声。
  • 死亡的边界：肉身停摆未必剧终；名字淡掉、记性被掏空，有时候比一下就疼死更难翻篇。
  • 力量的代价：权柄和活命多半拿平静、信任、敢说真话的胆子去换——潮退去沙滩上总得少一样东西。
  • 世界观锚：舞台埃尔西昂灰原三角（镇–旧殿–裂谷），势力条目与晦历年份详见《世界观设定集》十二则；主轴周目与九结局中英文对照另见 故事概览.md，《角色设定集》含五人执笔与五人×史据矩阵。
  • 线索叙事名（与终端 show_stats、story_text.txt 附录一致；脚本仍为 clue_*）：暗影徽记 · 缄卷裂真 · 七灯残阶 · 蜡誓遗册。
  • 羁绊显示规则（与终端 show_stats 同源）：四羁绊（mika_bond / cecil_bond / cassius_bond / lucien_bond）在主角未识认该角色（know_* = False）前，界面显示为「？？？羁绊」；识认后（首次见面对话完成）替换为「（角色名）羁绊」。识认触发点：米迦 Loop1 村口 · 塞西尔 Loop1 黄昏 · 卢西恩 Loop2 北街 · 卡修斯 Loop3 镇口。

命册节律 · 口授七夜（原「周内七日对话」）

  • 每周固定 7 次口授、每向 1 次分支；当周秘仪名依次为余烬权／缄影契／镜篆兆／裂契谶／回响钤／沉钟诰／阈扉仪（程序内 WEEK_LECTORY_ARCANUM），叙事氛围仍循余烬→暗线→镜纹→裂痕→回声→沉钟→门槛之阶
  • 每日根据当日角色切换专属选项与回应词库：米迦、塞西尔、卡修斯、卢西恩（见 VOICE_LIBRARY）
  • 选项肢统一带 角色名｜ 前缀（全角竖线），避免按日换人时玩家不知在回应谁；宽度由 ROLE_DIALOGUE_CONSTRAINTS.option_len 与 constrained_voice_block() 约束
  • 第一周：run_week_dialogues(..., 1) 前仍以体感与村口冲突为主；盐税告示与卢西恩／塞西尔／卡修斯三路长篇指名预告改在本章 Loop2 旁白首段读出，再衔接七日。run_week_dialogues(..., 1) 内仅以一句短接引扣回按日节奏。
  • know_* 门闸：第一周米迦村口＋首周口授、黄昏塞西尔；第二周口授前北街针线侧卢西恩观感入档；第三周镇口卡修斯——三至四周内线四人声纹齐备。口授题名与队内反应皆受 know_* 约束；未识时 resolve_oral_voice／ORAL_STANDIN_PROMPTS 代班
  • 日内小事件：每向前输出 「第 N 天」后插入 DAILY_MICRO_EVENT_RULES 旁白（七章×七日），再进入当日 focal。文案与存档识认对齐：在每条含多档旁白的日中，在满足「条目所需角色 ⊆ 玩家已 know_*」的前提下，优先选用「所需角色」条目最多的一条；未识认者可闻声见影得其习，不立姓名、不写与你比肩相认的私交，仅谣传埋线，避免信息越过玩家当时掌握。
  • Ren'Py：发布版在每轮主菜单前由 zzz_oral_bridge.rpy 运行时导入项目根的 text_runner.py 调用 run_week_dialogues；与终端口授条目逐字同源（缺模块时单行提示跳过）。分发请保留 text_runner.py 或与之一致的内嵌拷贝。
  • 达到最早触发日后执行周内暗骰；命中后主事件可提前进入，未命中则继续铺线
  • 周末结算按 soft / probe / press 倾向回写状态与关系，影响后续分支池与终局倾向

分支原则（哲学偏向版）

  • 不提供“最好/最坏”结局，只提供立场偏向
  • 每条结局必须同时给出即时收益与延迟后果
  • 提前收束会把代价前置，承担时间因此更早

判定骨架

  • 主检定：roll_check("坚韧与并肩", tenacity + sync, 12)
  • 暗骰：meta_roll()（逻辑层用数值；提前结局与裂缝 UI 的玩家可见文案不宣读点值）
  • 周内暗骰：hidden_week_gate(state, loop_no, day, mood)（决定主事件触发日）
  • 隐线：thread_mercy / obsession / erosion / revelation / scripture_mark
  • 关键状态：mika_knows_me / authority / tenacity
  • 对话约束层：ROLE_DIALOGUE_CONSTRAINTS + constrained_voice_block()（禁用词/句长阈值实时约束）

命运轴（Fate）

  • 接受既定结构，换稳定但失深层真相
  • 改写既定结构，换主动权与秩序风险
  • 被结构反噬，换短期生存与长期回环

个体轴（Self）

  • 保全边界，关系可能减弱
  • 交换边界，关系增强但自我磨损
  • 边界溶解，进入元层但失现实锚点

未知轴（Unknown）

  • 拥抱未知，扩大可达结局池
  • 利用未知，提升效率但代价前置
  • 拒绝未知，降低波动但冻结问题

黎明 / DAWN（命运轴）

石柱裂开了缝，风里墨味发干。你报名字——米迦念顺了些，倒也像是把一记迟晴天硬按回人间的样子。随后在边境镇上落脚，日子过得琐琐碎碎；更深那层就先搁着吧。

石柱裂开，像一份被执行完毕的裁决。米迦站在你面前，头发上全是碎石粉尘，眼神却慢慢沉回人间。风声被压成很薄的一层，像判令尾页上干透的墨。米迦说，收队。又说，像一场晚到的晴天，我到现在才把你名字念顺。你报上名字，奈尔。米迦回答，记住了，后面并肩。黎明落在台阶上，冷白而平整，像盖章后的纸面。后来你们在边境小镇住下，日子被拆成可核对的细目：租金、面包、巡夜铃。深层真相被装订成卷，收入档柜，暂不审理。

王座 / THRONE（命运轴）

你挤着人跨过裂口坐上黑王座——权贴在背上发凉，肩胛像烙了编号似的。次序拿稳了；至于还有没有人认真喊你……那是往后才慢慢算账的事。

你没有放手。裂口在你面前张开，像一道不可上诉的法令。你说，我还你。米迦被你推到裂缝另一边，他喊你的名字，却被石门裁成一段哑音。塞西尔朝你伸手，指间勾着一支黑羽箭，弓弦绷得像要割开风。石柱闭合了。黑暗往上升，像一张迟来太久的王座。你坐了上去。权力的寒光贴住肩背，像编号烙进骨里。从今往后，再也没有谁能把你从这里拖下来了。王座回应了你。它给你重量，也把你的影子钉在石上。你赢得了命运的控制权，也失去了被谁真正呼唤的资格。从此你化进秩序的夜间运转，名字逐寸褪去。

轮回 / CYCLE（命运轴）

村口风又打脸；告示换新纸罢了，落款折痕却仍卡在老地方。照旧点头照旧抬脚，流程像签字画押，撤不回呢。

轮回没有结束。你又一次站在村口。太阳还是那个太阳。米迦看向你，低声说，奇怪，我总觉得我们在这口风里站过很多次。风吹过公告牌，木板轻轻晃了一下。你点了点头。你知道自己还会陪他再走一遍。循环像钟表里那根细针，安静，固执，带着金属冷味。公告牌每天换上新纸，内容却总在同一段落处折返。你照旧点头，照旧抬脚，像按一份无法撤销的流程签下今天。

烛光 / CANDLELIGHT（个体轴）

你在崩塌里倒下，热气从肋骨间一点点被挤走。米迦的回应像旧誓反复发作，塞西尔转身去截断那条线。后来你的名字成了队伍里一处会痛的旧伤，每次触到都逼人站直。

你倒下了。石柱崩开时，碎屑像雪一样压下来，你先感觉到肋骨里的热气被一点点挤出去。你说，我没事。米迦回你，你每回都这么说。塞西尔把箭提起来：我去截断那条线。你留在掩体后。你听见自己的心跳越来越慢，像指腹下滑走的一缕脉搏。你闭上眼睛。有人在远处念你的名字，像抄写员读完一行旧文。纸页翻过去了。后来他们还会提起你，在火光最弱的时候，在每一次不得不做选择的时候。你的名字变成一处旧伤。每次触到，都会痛，也都会让人站直。

飞升 / ASCENSION（个体轴）

你握米迦那只手——光爬上指缝，石壁神殿一层层退去。新路是踏上了，可要记住的句子却一天比一天薄：只剩手心那点热还认得。

誓约分摊 / COVENANT（个体轴）

朝圣者 / PILGRIM（未知轴）

审判 / JUDGEMENT（未知轴）

门只开到半掌，七盏灯随后依次亮起——判词先在你掌心里淌开，才去照旁人。你真带走了一部分；骨血名声却粘在门板深处抠不下来。

作者 / AUTHOR（未知轴）
