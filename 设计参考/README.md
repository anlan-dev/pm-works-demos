## 🎯 收集建议

### 截图什么？

1. **首页布局**：信息如何组织，导航如何设计
2. **核心流程**：用户完成任务的完整路径
3. **交互细节**：动画、转场、手势反馈
4. **视觉风格**：配色、字体、图标、间距
5. **空状态**：没有数据时的展示
6. **错误状态**：网络错误、加载失败

### 如何命名？

```
[APP名称]_[页面类型]_[特点].png

示例：
- 滴答清单_首页_极简布局.png
- 小红书_详情页_瀑布流.png
- Keep_运动页_数据可视化.png
```

## 📱 截图工具推荐

### iOS
- **系统截图**：电源键 + 音量上键
- **长截图**：Safari 支持整页截图
- **录屏**：控制中心 → 屏幕录制

### Android
- **系统截图**：电源键 + 音量下键
- **长截图**：截图后点击「滚动截图」
- **录屏**：下拉通知栏 → 屏幕录制

### 电脑端
- **网页截图**：浏览器开发者工具 → 设备模式
- **录屏**：OBS Studio（免费）

## 💡 使用方法

1. **浏览 APP 时随手截图**：看到好的设计就截
2. **按分类存放**：方便查找
3. **标注亮点**：在图片名或备注里写为什么好
4. **定期整理**：每周花 10 分钟整理新截图
5. **面试前复习**：看看别人的优秀设计，找灵感
