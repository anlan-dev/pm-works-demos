## 一、布局系统

### 1.1 栅格系统（12列）

```css
.grid {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 16px;
}

/* 常用布局 */
.grid-2 { grid-template-columns: repeat(2, 1fr); }
.grid-3 { grid-template-columns: repeat(3, 1fr); }
.grid-4 { grid-template-columns: repeat(4, 1fr); }

/* 响应式 */
@media (max-width: 768px) {
  .grid-2, .grid-3, .grid-4 {
    grid-template-columns: 1fr;
  }
}
```

### 1.2 响应式断点

| 断点 | 宽度 | 设备 | 栅格 |
|------|------|------|------|
| **xs** | < 576px | 手机竖屏 | 1列 |
| **sm** | ≥ 576px | 手机横屏 | 2列 |
| **md** | ≥ 768px | 平板 | 3列 |
| **lg** | ≥ 992px | 小桌面 | 4列 |
| **xl** | ≥ 1200px | 桌面 | 4列 |
| **xxl** | ≥ 1400px | 大屏 | 4列 |

### 1.3 容器规范

```css
/* 主内容区 */
.container {
  max-width: 1120px;
  margin: 0 auto;
  padding: 0 24px;
}

/* 窄内容区（文章/表单） */
.container-sm {
  max-width: 800px;
  margin: 0 auto;
  padding: 0 24px;
}

/* 宽内容区（数据大屏） */
.container-lg {
  max-width: 1400px;
  margin: 0 auto;
  padding: 0 32px;
}
```

### 1.4 常用布局模式

#### A. 两栏布局（侧边栏 + 主内容）


#### B. 三栏布局（左导航 + 内容 + 右面板）

```css

@media (max-width: 1200px) {

    grid-template-columns: 240px 1fr;
```

#### C. 卡片网格

```css

  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));

```

#### D. Dashboard 布局

```css

  display: grid;

@media (max-width: 768px) {

    grid-template-columns: 1fr;
  }
  .dashboard .stats,
  .dashboard .chart-main,
  .dashboard .chart-side {
    grid-column: span 1;
  }
}
```

## 二、组件规范

### 2.1 按钮（Button）

#### 尺寸

| 类型 | 高度 | 内边距 | 字号 | 圆角 |
|------|------|--------|------|------|
| **Small** | 32px | 8px 16px | 0.75rem | 8px |
| **Medium** | 40px | 10px 20px | 0.85rem | 10px |
| **Large** | 48px | 14px 28px | 0.95rem | 12px |

#### 样式

```css
/* Primary */
.btn-primary {
  background: var(--primary);
  color: #fff;
  border: none;
  font-weight: 600;
  transition: all 0.2s ease;
}

.btn-primary:hover {
  background: var(--primary-dark);
  box-shadow: 0 4px 12px var(--primary-glow);
}

/* Secondary */
.btn-secondary {
  background: transparent;
  color: var(--primary);
  border: 1px solid var(--primary);
}

/* Ghost */
.btn-ghost {
  background: transparent;
  color: var(--text2);
  border: none;
}

.btn-ghost:hover {
  background: var(--surface2);
}

/* Danger */
.btn-danger {
  background: var(--danger);
  color: #fff;
  border: none;
}
```

#### 状态

| 状态 | 视觉变化 | 用途 |
|------|----------|------|
| **Default** | 正常显示 | 默认状态 |
| **Hover** | 颜色加深 + 阴影增强 | 暗示可点击 |
| **Active** | 颜色更深 + 位移下沉 | 按压反馈 |
| **Disabled** | 透明度50% + 禁止光标 | 不可用状态 |
| **Loading** | 替换为Spinner | 操作中 |

## 四、响应式设计

### 4.1 移动端优先原则

1. **内容优先**：移动端只展示核心内容
2. **触控友好**：所有可点击区域 ≥ 44px
3. **手势支持**：下拉刷新、左右滑动
4. **性能优先**：减少动画、延迟加载

### 4.2 响应式策略

| 元素 | 桌面端 | 移动端 |
|------|--------|--------|
| **导航** | 水平导航栏 | 汉堡菜单 |
| **侧边栏** | 固定显示 | 抽屉式/隐藏 |
| **卡片网格** | 3-4列 | 1-2列 |
| **表格** | 完整显示 | 横向滚动/卡片化 |
| **字体** | 标准尺寸 | 缩小10-20% |
| **间距** | 标准间距 | 缩小25% |

### 4.3 媒体查询模板

```css
/* Mobile First */
.element {
  /* 基础样式（移动端） */
  padding: 16px;
}

/* Tablet */
@media (min-width: 768px) {
  .element {
    padding: 24px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .element {
    padding: 32px;
  }
}

/* Large Desktop */
@media (min-width: 1400px) {
  .element {
    padding: 40px;
  }
}
```
