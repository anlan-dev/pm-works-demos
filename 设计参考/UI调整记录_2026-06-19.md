## 三、跨境电商 Demo 专项调整

### 3.1 按钮交互修复

| Bug | 根因 | 修复方案 |
|-----|------|----------|
| 所有按钮失效 | IIFE 提前关闭 | 删除多余 `})();` |
| AI浮动按钮失效 | handler 冲突 | 独立 getElementById + null 检查 |
| 分享/复制无反应 | clipboard API 不可用 | 内联 onclick + execCommand 兜底 |

### 3.2 新增动态功能

| 功能 | PM 决策理由 | 实现方式 |
|------|-------------|----------|
| 实时倒计时 | 限时拼团核心体验 | setInterval 每秒更新 |
| 实时动态滚动 | 社交证明 | 滚动数组 + 3秒轮播 |
| 拼团进度动画 | 降低决策犹豫 | CSS width 动画 |
| 库存递减 | 稀缺性暗示 | 随机间隔递减 |
| 按钮涟漪效果 | 点击反馈微动效 | CSS @keyframes ripple |
| AI推荐卡片切换 | 展示选品能力 | Tab 切换 |
