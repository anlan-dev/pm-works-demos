## 一、色彩体系

### 1.1 通用色彩语义（跨项目复用）

| 语义 | 色值 | 用途 | 适用场景 |
|------|------|------|----------|
| **Primary** | `#0071e3` | 主操作、链接、强调 | 工具类/效率类产品 |
| **Success** | `#10b981` | 成功状态、正向增长 | 数据看板、操作反馈 |
| **Warning** | `#f59e0b` | 警告状态、需注意 | 表单验证、提示信息 |
| **Danger** | `#ef4444` | 错误状态、删除操作 | 错误提示、危险操作 |
| **Info** | `#6366f1` | 信息提示、次要强调 | 说明文字、辅助信息 |

### 1.2 各项目色彩方案

#### A. 浅色主题（工具效率类）

```
背景层：
  --bg: #f8fafc          /* 主画布 浅灰 */
  --surface: #ffffff      /* 卡片 白色 */
  --surface2: #f1f5f9     /* 次级表面 浅蓝灰 */

文字层：
  --text: #0f172a         /* 主文字 深蓝黑 */
  --text2: #475569        /* 次要文字 中灰 */
  --text3: #94a3b8        /* 辅助文字 浅灰 */

边框层：
  --border: #e2e8f0       /* 默认边框 */
  --border-hover: #cbd5e1 /* 悬停边框 */
```

**适用项目**：01-finance-dashboard, 02-health-tracker, 03-saas-pricing

#### B. 新拟物主题

```
背景层：
  --bg: #e6e6e6           /* 经典新拟物灰 */
  --surface: #e6e6e6      /* 同色凸起 */

阴影层：
  --shadow-light: -8px -8px 16px rgba(255,255,255,0.66)
  --shadow-dark: 8px 8px 16px rgba(0,0,0,0.12)
```

**适用项目**：04-pet-vaccine, 07-prompt-library

#### C. 暗色主题（沉浸式）

```
背景层：
  --bg: #050507           /* 深色画布 */
  --bg2: #0c0c10          /* 次级背景 */
  --card: #111115          /* 卡片 */
  --card-hover: #16161c    /* 卡片悬停 */

文字层：
  --text: #f0f0f5          /* 主文字 亮白 */
  --text2: #8b8b9e         /* 次要文字 中灰 */
  --text3: #5a5a6e         /* 辅助文字 暗灰 */

强调色：
  --accent: #a78bfa        /* 紫色强调 */
  --accent2: #60a5fa       /* 蓝色辅助 */
  --accent3: #34d399       /* 绿色点缀 */
```

**适用项目**：08-glassmorphism, 09-cyberpunk

#### D. 日式极简主题

```
背景层：
  --bg: #faf8f5            /* 米白画布 */
  --surface: #ffffff       /* 白色卡片 */

文字层：
  --text: #2c2c2c          /* 主文字 深灰 */
  --text2: #666666         /* 次要文字 */

强调色：
  --accent: #c4553a        /* 朱红 单色强调 */
```

**适用项目**：10-japanese-minimal

#### E. 粗野主义主题

```
色彩：
  --bg: #ffffff            /* 纯白背景 */
  --text: #000000          /* 纯黑文字 */
  --accent: #ff0000        /* 红色强调 */
  --border: #000000        /* 黑色边框 */
```

**适用项目**：11-brutalist

## 二、字体系统

### 2.1 字体栈（全局统一）

```css
font-family: 'Inter', 'Noto Sans SC', system-ui, -apple-system, sans-serif;
```

| 字体 | 角色 | 来源 | 用途 |
|------|------|------|------|
| **Inter** | 主字体 | Google Fonts | 英文、数字、UI元素 |
| **Noto Sans SC** | 中文字体 | Google Fonts | 中文内容 |
| **system-ui** | 回退字体 | 系统 | 加载失败时的降级 |

### 2.2 字体层级（Type Scale）

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| **Display** | 2.4-4rem | 900 | 1.1 | Hero 标题、大数字 |
| **H1** | 1.5-2rem | 800 | 1.2 | 页面主标题 |
| **H2** | 1.25-1.5rem | 700 | 1.3 | 区块标题 |
| **H3** | 1-1.15rem | 600 | 1.4 | 卡片标题 |
| **Body** | 0.875-1rem | 400 | 1.6 | 正文内容 |
| **Caption** | 0.75-0.85rem | 500 | 1.5 | 标签、说明文字 |
| **Overline** | 0.65-0.72rem | 600 | 1.4 | 大写标签、辅助信息 |

### 2.3 字重使用规范

| 字重 | 数值 | 用途 |
|------|------|------|
| **Light** | 300 | 大号展示文字（慎用） |
| **Regular** | 400 | 正文、段落 |
| **Medium** | 500 | 次要强调、导航链接 |
| **SemiBold** | 600 | 标签、小标题、按钮 |
| **Bold** | 700 | 区块标题 |
| **ExtraBold** | 800 | 主标题、数据数字 |
| **Black** | 900 | Hero标题、大数字 |

### 2.4 数字显示规范

```css
/* 等宽数字，避免跳动 */
font-variant-numeric: tabular-nums;

/* 紧凑字距用于大数字 */
letter-spacing: -0.04em;

/* 金额/百分比等数据 */
font-weight: 800;
font-variant-numeric: tabular-nums;
```

## 四、圆角规范

| Token | 值 | 用途 |
|-------|-----|------|
| `--r-sm` | 8px | 小按钮、标签、输入框 |
| `--r-md` | 12px | 按钮、小卡片 |
| `--r-lg` | 16px | 大卡片、容器 |
| `--r-xl` | 20px | 特殊卡片、模态框 |
| `--r-full` | 999px | 药丸按钮、头像、徽章 |

### 圆角使用原则

- **一致性**：同类型元素使用相同圆角
- **层级关系**：外层圆角 ≥ 内层圆角
- **品牌调性**：
  - 工具类：12-16px（专业感）
  - 社交类：16-20px（友好感）
  - 极简类：4-8px（克制感）
  - 粗野主义：0px（原始感）

## 五、阴影规范

### 5.1 浅色主题阴影

```css
--shadow-sm: 0 1px 3px rgba(0,0,0,0.04);
--shadow: 0 1px 3px rgba(0,0,0,0.04), 0 8px 24px rgba(0,0,0,0.04);
--shadow-lg: 0 4px 12px rgba(0,0,0,0.04), 0 16px 48px rgba(0,0,0,0.06);
```

### 5.2 暗色主题阴影

```css
--shadow: 0 4px 24px rgba(0,0,0,0.3);
--shadow-lg: 0 8px 48px rgba(0,0,0,0.5);
```

### 5.3 新拟物阴影

```css
--shadow-neu: 8px 8px 16px rgba(0,0,0,0.12), -8px -8px 16px rgba(255,255,255,0.66);
```

### 5.4 交互态阴影

```css
/* 悬停增强 */
.card:hover {
  box-shadow: var(--shadow-lg);
  transform: translateY(-2px);
}

/* 按压减弱 */
.button:active {
  box-shadow: var(--shadow-sm);
  transform: translateY(0);
}
```

## 六、设计规范检查清单

### 色彩检查

- [ ] 文字与背景对比度 ≥ 4.5:1（WCAG AA）
- [ ] 主色不超过 3 个（Primary + 1-2 Accent）
- [ ] 状态色语义一致（Success=绿、Warning=黄、Danger=红）
- [ ] 暗色模式下文字亮度 ≥ 85%

### 字体检查

- [ ] 全局字体栈统一：Inter + Noto Sans SC
- [ ] 标题层级清晰（H1 > H2 > H3）
- [ ] 数字使用 tabular-nums 避免跳动
- [ ] 行高适配：标题 1.1-1.3、正文 1.6

### 间距检查

- [ ] 使用 4px 基数的间距系统
- [ ] 同类元素间距一致
- [ ] 容器内边距统一（24px 或 32px）
- [ ] 触控区域 ≥ 44px（移动端）
