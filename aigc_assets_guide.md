# AIGC素材管理

本文件夹用于存放AI生成的视觉小说素材，包括角色立绘、背景图、UI元素等。

## 文件夹结构

```
aigc_assets/
├── characters/          # 角色立绘
│   ├── base/           # 基础角色图
│   ├── emotions/       # 表情变体
│   └── poses/          # 姿势变体
├── backgrounds/         # 背景场景图
├── ui/                  # UI元素
│   ├── buttons/        # 按钮
│   ├── frames/         # 对话框框架
│   └── icons/          # 图标
├── effects/             # 特效素材
└── raw/                 # 原始生成图（未处理）
```

## 推荐工具

### 本地部署（免费）
- **ComfyUI + VNCCS**：专为视觉小说设计，推荐使用
- **Stable Diffusion + ControlNet**：通用方案，灵活性高

### 在线服务
- **Midjourney**：质量高，月费$10起
- **Leonardo.ai**：免费额度，游戏资产功能专门优化
- **Tensor.Art**：免费，社区模型丰富

## 工作流程建议

1. **角色设计**：先生成基础角色图，确定外观风格
2. **表情生成**：基于基础图生成不同表情变体
3. **背景生成**：统一风格生成场景背景
4. **UI设计**：生成对话框、按钮等UI元素
5. **精修处理**：使用Photoshop/GIMP进行细节修正

## 文件命名规范

- 角色：`角色名_表情_姿势.png`（如：`lihua_smile_01.png`）
- 背景：`场景_时间_天气.png`（如：`classroom_morning_sunny.png`）
- UI：`元素类型_名称.png`（如：`button_start.png`）

## 注意事项

- 保持美术风格统一，使用固定的提示词模板
- 记录生成参数（模型、种子、提示词等），便于复现
- 定期备份重要素材
- 注意版权问题，确认使用的模型允许商用
