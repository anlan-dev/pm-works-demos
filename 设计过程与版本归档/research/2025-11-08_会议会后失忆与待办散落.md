# 访谈纪要（脱敏）· 会议反刍舱 — 会后失忆与待办散落

| 项目 | 内容 |
|------|------|
| **内部编号** | P-2025-11-08-M |
| **关联产品** | 会议·反刍舱 |
| **形式** | 协作者半结构访谈（自用场景延伸） |
| **时长** | 约 30 分钟 |

## 1. 场景

- 受访者每周 **≥2 场**线上站会，纪要散落在 IM 与备忘录；**会上认领的 action item 常无人更新状态**。

## 2. 原话摘录（脱敏）

<details>
<summary>💬 展开查看匿名用户原话（已脱敏）</summary>

- 「我最怕的是**会开完了群里还在重复问谁做什么**。」  
- 「如果待办能**有个状态**，至少我知道这事是做了还是卡住了。」  
- 「录音**不想上传公网**，能不能先本地转一圈？」

</details>

## 3. 结论 → 产品决策（与 PRD / 作品集一致）

| 点 | 决策 |
|----|------|
| 范围 | **不做**大而全协作台，先做「转写 → 待办 → 状态流 → 复盘」 |
| 隐私 | 保留 **本地 Whisper** 路径 narrative |
| 验证 | **6～8 场**演练 + **4 名**协作者跟评；折线为**示意指数** |

## 4. 样本边界（面试主动交代）

- 非企业级并发；结论用于 **Demo 迭代节拍**与**方法展示**。
