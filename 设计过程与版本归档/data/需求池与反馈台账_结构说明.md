## 3. 单条工单建议字段

| 字段 | 说明 |
|------|------|
| `ticket_id` | 如 FD-2026-036 |
| `title` | 短标题 |
| `project` | pet / meeting / 跨境 / pixelpick / mistlamp / branchtree / demohub / 全项目 |
| `status` | closed / active / triage |
| `theme` | 原有 + 新增（见上表） |
| `source` | interview / usability / dogfood / self |
| `created_at` / `closed_at` | 日期 |
| `release_note` | 关联版本标签 |
