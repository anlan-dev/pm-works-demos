# 表格模板 · 规划用（非 Figma 官方导入）

将下表复制到 Numbers / Excel / Google Sheets，用于**自拟 Frame 列表**；在 Figma 中仍需**手动**创建 Frame（或配合自建插件）。

模板文件：`figma_frame_import_template.csv`（UTF-8）

列说明：

| 列 | 含义 |
|----|------|
| project_id | 与 `manifest.json` 中 id 一致 |
| frame_name | Figma 内 Frame 命名建议 |
| width | 像素 |
| height | 像素 |
| hi_fi_file | 相对 `demo/work` 的 HTML |
| notes | 可选备注 |
