# 附录 · 高频追问 · 应答要点

## 1. 「你的样本这么小，结论能信吗？」

**答**：这是** Demo 阶段** deliberate 的选择——用固定口径把「前后是否同向」说清楚，而不是假装有大厂流量。若进入实习，会把同一套口径迁到更大样本与实验平台上，补**功效与显著性**。

**补充**：小样本深访的价值在于**挖出痛点本质**，而非统计显著性。20人的访谈足以发现"多宠隔离提醒"这个核心需求，大样本问卷则用于验证这个需求的普遍性。

## 2. 「示意指数是不是编的？」

**答**：不是拍脑袋装饰图，而是把**多项信号按预设权重合成**，基线归一为 100，只用于**相对比较**；具体权重与字段写在个人实验笔记里，可展开。

**补充**：权重是事先写死的，不是事后调整来美化数据。比如宠伴的提醒配置意愿指数，输入包括：任务完成率（权重0.4）、提醒配置量表（权重0.3）、回访意愿（权重0.3）。

## 3. 「需求和竞品分析在哪？」

**答**：宠伴 case 在作品集有**竞品对比矩阵**；书面扩展在对应 **PRD「非目标 / 关键决策」** 两节。

**补充**：竞品分析我做了3个维度：功能覆盖（是否支持多宠）、隐私策略（是否本地优先）、使用门槛（是否需要注册）。最终选择"轻量健康记录册"定位，是因为竞品要么太重（社区+商城），要么隐私策略不符合用户预期。

## 4. 「你怎么证明用户真的觉得好用？」

**答**： triangulation——**任务**（能不能完成）、**时间感受**（走查/问卷）、**行为**（本地日志事件），三者不对齐会继续迭代而不是写漂亮结论。

**补充**：比如宠伴的"自动算有效期"功能，任务完成率100%（都能完成），主观耗时缩短60%（觉得更快了），本地日志显示该功能使用频次提升（真的在用）。三个维度一致，才认为这个功能有效。

## 5. 「AI 在你的流程里到底干什么？」

**答**：缩短「写 Prompt → 出原型 → 收反馈」的周期；AI **不替代**口径定义与实验设计；Whisper 等是**可选路径**，敏感场景走本地。

**补充**：具体来说，AI在我的项目中承担3类工作：
1. **内容生成**：会议舱用DeepSeek分析转写文本，提取待办
2. **个性化推荐**：冷启动用AI生成定制化诊断和计划
3. **辅助决策**：住值用AI顾问给出租房建议

但核心的产品决策——做什么、不做什么、怎么验证——都是我基于用户研究做出的。

## 6. 「如果让你三个月做成真产品，你第一步做什么？」

**答**：冻结 v1 指标与事件字典 → 接最小埋点/导出 → 上一组**可随机**的运行实验（视资源）→ 把 Demo 里已验证的**主路径**产品化，延迟非核心模块。

**补充**：具体来说：
- **第1周**：冻结指标口径，定义5个核心事件（建档、提醒设置、记录提交、OCR使用、导出）
- **第2-3周**：接入埋点SDK，上线最小可用版本
- **第4-6周**：招募100名用户，随机分组测试提醒策略
- **第7-8周**：分析数据，迭代核心路径
- **第9-12周**：产品化主路径，延迟非核心功能

## 7. 「你做过最复杂的项目是什么？」

**答**：**住值·租房生命成本计算器**。它实现了贪心预算分配算法，把通勤时间、健康损耗、交通费用等隐性成本量化，输出双层Bmax公式计算的房租预算上限。技术上，我用JavaScript实现了算法逻辑，用Chart.js做可视化，还集成了AI顾问给出个性化建议。

**补充**：这个项目的难点在于**把复杂计算逻辑产品化**——用户不需要理解算法，只需要输入几个关键参数，就能得到可理解、可决策的结果。

## 8. 「你怎么处理用户反馈和需求优先级？」

**答**：我用**需求池+状态流**管理。每条反馈记录：单号、项目、主题、状态（已闭环/进行中/待评审）、优先级、关联版本。优先级判断标准：影响核心路径的优先、高频问题优先、低成本高收益的优先。

**补充**：比如宠伴的需求池有17条工单，我优先闭环了"提醒灵活性"和"OCR"两类问题，因为它们直接影响用户是否愿意继续使用。

## 9. 「你对跨境电商AI Agent有什么理解？」

**答**：我认为核心价值是**降低试错成本**。跨境电商需要快速测试不同市场、不同素材、不同投放策略，AI Agent可以自动化完成"测试→分析→优化"的循环，把原本需要一周的迭代缩短到一天。

**补充**：具体来说：
- **AI一键建站**：用AI生成产品描述、图片、落地页，快速测试不同市场
- **自动化投流监测**：用AI实时监控广告效果，自动调整出价和素材
- **广告素材Agent洞察**：用AI分析竞品素材，提取高转化特征
- **视频素材混剪优化**：用AI自动剪辑、配音、字幕，生成多版本素材

这些场景都需要快速试错，和我的产品思维很契合。
