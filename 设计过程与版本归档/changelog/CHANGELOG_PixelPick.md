# PixelPick · AI 影像精选 — 迭代日志

> 记录每个版本的关键决策、用户反馈与验证结果。

---

## v1.0 (2026-06-26) — MVP 上线

**决策依据**：设计师/摄影师选图流程繁琐——需要手动浏览、对比、打分、筛选，缺乏 AI 辅助。

**核心改动**：
- 批量上传（拖拽+点击，支持 JPEG/PNG/WebP）
- 暗房接触印相（Contact Sheet）网格展示
- 前端模拟 4 维度评分（构图/曝光/对焦/色彩）
- Best 精选自动标记（★ + 蓝色边框 + 锯齿切角）
- 放大镜悬停预览波纹
- 5 步胶片齿孔进度条
- 精选横幅（最佳图片 + 评分 + 入选理由）

**设计隐喻**：暗房接触印相 + Apple HIG 暗色规范
- 炭黑画布 `#000000` 模拟暗房环境
- 琥珀蓝 `#0071e3` 模拟暗房安全灯光
- 胶片齿孔进度条暗示摄影工艺流程

**验证结果**：
- 纯前端实现，零后端依赖，响应 < 100ms
- Contact Sheet 最多支持 50+ 图片流畅展示
- Best 算法为前端模拟，非真实 AI 推理（需在产品说明中标明）

**技术选型**：
- FileReader API 读图
- Canvas API 缩略图生成
- CSS Grid 自适应 Contact Sheet
- 纯前端评分算法（不依赖 AI API）

---

## v1.1 (2026-06-27) — 暗房模式扩展

**决策依据**：用户反馈希望更多「暗房感」——真实暗房的红色 safelight 模式和对比度增强。

**核心改动**：
- 新增「暗房红」主题（safelight 红色调）
- 放大镜对比模式（分屏对比两张图片）
- 批量 BEST 标记（手动覆写 AI 评分）
- 导出优化（按评分筛选 + 批量下载重命名）

**验证结果**：
- 暗房红主题增强场景沉浸感
- 分屏对比提升选图效率
