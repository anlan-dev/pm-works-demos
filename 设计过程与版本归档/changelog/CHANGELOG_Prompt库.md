# Prompt Library — 迭代日志

> 记录每个版本的关键决策、用户反馈与验证结果。

---

## v1.0 (2025.12) — MVP 上线

**决策依据**：产品经理需要管理大量 AI Prompt。

**核心改动**：
- Prompt 管理（创建/编辑/删除）
- 分类系统（多级分类）
- 一键复制
- 本地存储（零服务器）

**验证结果**：
- 自用 + 种子用户试用
- 一键复制使用率约 80%

---

## v1.1 (2026.01) — 搜索 + 标签

**决策依据**：Prompt 数量增多后难以查找。

**核心改动**：
- 搜索过滤（关键词/分类/标签）
- 标签系统（高频标签推荐）
- 使用统计

**验证结果**：搜索功能使用率约 60%。

---

## v1.2 (2026.02) — 导入/导出 + 持续维护

**决策依据**：用户需要跨设备同步和备份。

**核心改动**：
- JSON 导入/导出
- 持续维护机制
- 使用统计可视化

**验证结果**：导入/导出功能使用率约 50%。

---
