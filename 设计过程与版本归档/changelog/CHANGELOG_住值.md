# 住值·租房生命成本 — 迭代日志

> 记录每个版本的关键决策、用户反馈与验证结果。

---

## v1.0 (2025.12) — MVP 上线

**决策依据**：租房时只看租金，忽略隐性成本（时间/交通/生活）。

**核心改动**：
- 租房记录（租金/押金/水电/交通）
- 成本计算（多维度模型）
- 决策建议（基于成本对比）
- 本地存储（零服务器）

**验证结果**：
- 种子用户试用，反馈「终于看清真实成本了」
- 决策建议采纳率约 60%

---

## v1.1 (2026.01) — 数据看板 + 对比

**决策依据**：用户需要对比多个租房选项。

**核心改动**：
- 数据看板（Chart.js 可视化）
- 多租房对比分析
- 成本构成饼图

**验证结果**：对比功能使用率约 70%。

---

## v1.2 (2026.02) — 导出 + 模板

**决策依据**：用户需要导出租房报告。

**核心改动**：
- 导出 JSON/报告
- 租房模板（常见场景）
- 历史记录查看

**验证结果**：导出功能使用率约 40%。

---

## v1.3 (2026.06) — UI 风格迁移 + 按钮交互排查

**决策依据**：全项目UI审查，统一迁移至标准设计系统。

**核心改动**：
- 从 Neumorphic 迁移至白卡片（#ffffff）+ 浅灰画布（#f5f5f7）
- 统一字体/主色/阴影/圆角
- 按钮交互排查，确认无IIFE/clipboard同类bug

**验证结果**：语法检查通过，按钮事件绑定检查通过。
